# Seasonal Temperature Heatmap

This project is a React + D3 data visualization app that displays average weekly temperatures for selected cities across 2025 in an interactive heatmap.

It is part of the **D3 Loves React** course and serves as a hands-on project for learning how to fetch and transform live data from an API before visualizing it with D3 in React.

## What this project covers

- Fetching data from the [Open-Meteo](https://open-meteo.com/) archive API
- Transforming and aggregating API responses with D3 (`rollups`, `mean`)
- Building a responsive heatmap with scales, axes, and a color legend
- Adding hover tooltips for exploring individual values

## Run locally

Install dependencies and start the development server:

```bash
npm install
npm run dev
```
